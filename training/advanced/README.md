# Advanced path (~2 hours)

For release engineers.

1. **Promotion gates** in depth — maintenance windows (config + ad-hoc),
   QA/release Go-No-Go sign-off, change-request codes (JIRA). Lab 08 with
   [bank-api](../sample-apps/bank-api).
2. **Ref-pinned rings** — acc always ships `release` ([ai-chat](../sample-apps/ai-chat)).
3. **Canary & blue/green** — Lab 09 with [shopping-cart](../sample-apps/shopping-cart).
4. **AI failure diagnosis** — reading and explaining failed jobs.
5. Lab 10 — a full production incident, end to end.

**You'll be able to:** run governed production promotions, choose progressive-
delivery patterns, and drive incident response. Next:
[enterprise](../enterprise).
